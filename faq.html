Frequently asked questions

Does it work in every browser?

In principle, it works in every modern up-to-date browser. However, there are some known issues with the implementation of certain HTML5 features. That is why we recommend using the most current version of Google Chrome or Mozilla Firefox.

We haven't tested the app on mobile browsers and the interface is not for touch interaction. In principle, it should work, but it is unsupported.

Does it work without an internet connection?

Yes. After visiting CMDI Maker for the first time, the app is automatically stored in your browser.
If you now call the url cmdi-maker.uni-koeln.de, the app opens even without being online.

Why is CMDI Maker not reachable when the internet connection is offline?

There are basically two options:
1. You are not using the same browser on the same computer you used before. The data is stored in the browser itself, and is not available in other browsers on the same computer or on different computers.
2. Your browser settings do not allow saving application data on your machine or your browser deletes saved data on exit.

All my data is gone! What happened?

There are basically two options:
1. You are not using the same browser on the same computer you used before. The data is stored in the browser itself, and is not available in other browsers on the same computer or on different computers.
2. Your browser settings do not allow saving application data on your machine or your browser deletes saved data on exit.

We recommend to additionally backup your data by exporting your project as a CMP file (Hamburger Menu > Export Project to File).

Which profile should I use?

This depends on the intended archive for your data. Contact the archive and ask which metadata variants it accepts.
